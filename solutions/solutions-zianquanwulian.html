    自安全物联感知网解决方案
    随着物联感知网的迅速发展及基础设施通信系统的IP化，大量 IP 摄像机、抓拍器、RFID 等前端设备已经大规模部署在城市的各个角落。物联感知网前端设备大量分散在无人值守的环境下，人为监管困难，极易被黑客利用，进而渗透到整个网络,美襄科技聚焦物联感知网终端准入控制技术、L2~7 层白名单技术等多种关键技术的研究，可以对整个物联感知网前端 IP 设备和传输的流量进行精确管控，只有通过认证的设备才允许接入，只有合法的应用才允许在网络中传输，从而防范非法私接、设备仿冒、非法扫描、DDoS攻击等问题。

    IOT STATUS AND SECURITY RISKS

    物联感知网现状及安全风险

    非法私接、仿冒难以防范

    前端设备分布点多面广，非法私接、仿冒难以防范，安全风险大。

    专网边界增多 安全风险增多

    专网整合了政府面、社会面等视频资源，导致边界增多，风险大大增加。

    核心数据及其容易受攻击

    物联感知网内数据资源价值巨大，针对核心数据的攻击行为将增多。

    IP地址使用容易产生混乱

    各类业务混杂在一起，易带来内部安全问题，也方便入侵者扩大攻击范围。

    资产状态无法准确统计

    主要依托平台级联，数量统计不准确、信息采集不准确、数据更新不及时。

    资产运维存在安全隐患

    运维电脑违规外联，外包服务人员非法获取并泄露关键数据。

    SOLUTION DEPLOYMENT

    方案部署

    美襄科技可为用户提供涵盖网络核心、物联网应用安全控制系统及安全接入交换机等全系列物联承载网产品及解决方案。通过部署DAC（Device Application Controller，物联网应用安全控制系统），实现设备接入可信、行为可控。网管平台与整网设备联动，同步下发策略，并提供资产与风险管理工具，实现整网接入、资源保护及威胁控制。

    Permalink

    准入认证及应用控制

    美襄科技针对物联网的设备与应用特点，采用白名单技术框架，首先只允许授信设备接入网络，对于未通过认证的设备进行实时阻断，不允许其接入网络，并进行告警；其次识别合法设备的网络访问行为，只允许其传输与预先确定的应用相关的数据。

    Permalink

    灵活部署

    可根据部署情况灵活选择产品系列：大型网络采用可扩展架构，部署于物联网汇聚层；小型网络采用盒式架构，部署适当下移；在视频监控场景下，单控制点最大可达32000路4M码流IPC准入控制能力。
